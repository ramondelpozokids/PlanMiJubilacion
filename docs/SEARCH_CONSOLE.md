## Ya listo en el código
- Verificación HTML: `public/google02f28f347260647f.html`
- Sitemap: `https://plan-mi-jubilacion.vercel.app/sitemap.xml`
- Robots: `https://plan-mi-jubilacion.vercel.app/robots.txt`

## Qué hacer en Google Search Console
1. Propiedad URL-prefix: `https://plan-mi-jubilacion.vercel.app`
2. Si pide verificación: el archivo HTML ya está publicado (método archivo HTML).
3. **Sitemaps** → añadir: `sitemap.xml`
4. Cuando tengas dominio propio (p. ej. `planmijubilacion.es`), añade esa propiedad y actualiza `NEXT_PUBLIC_APP_URL` en Vercel.

## Meta verification opcional
En Vercel puedes definir `GOOGLE_SITE_VERIFICATION=token` (sin comillas) si prefieres la etiqueta meta en lugar del HTML.
